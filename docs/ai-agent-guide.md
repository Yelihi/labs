# AI Agent Guide

## Role

The AI agent may:

- create lab scaffolds
- write README drafts
- write plan drafts
- create React/Vite demo code
- create Playwright tests
- summarize Playwright results
- export notes to Obsidian format

## The AI agent must not:

- invent browser compatibility results
- invent performance numbers
- mark a lab as done without manual confirmation
- claim Safari/Edge behavior without actual test or manual note
- overwrite manual-check.md without preserving existing content

## Human-owned decisions

The human must decide:

- the topic is worth studying
- the experiment question is valid
- the visual result is correct
- the browser difference matters
- the final practical judgment is acceptable
