# PLAN - Domain Develop With Functional Programing

## Experiment Question

이번 실습은 함수형 프로그래밍을 통해 typescript 를 기반으로 하여 도메인 주도 설계를 study 하는 실습이며, 기존 도메인 주도 설계는 주로 객체지향 프로그래밍이 기반이 되었는데, 이와 반대되는 개념이라 궁금증을 해소하고자 한다.

## Hypothesis (가설)

- 객체지향 프로그래밍에서는 상태를 가진 객체가 비즈니스 로직을 수행하는 반면, 함수형 프로그래밍에서는 상태를 가지지 않는 함수가 비즈니스 로직을 수행한다. 

## Implementation Plan

- 서적을 쭉 읽으면서 전체적인 내용을 먼저 스캐닝 한다.
- 설계 파트부분을 요약하여 md 작성후 skills 로 변환한다.
- 해당 설계 파트 부분을 통해 실제 프로젝트의 설계를 진행해본다.
- 함수형 프로그래밍 내 구현 부분을 읽고 약간의 감을 보는것으로 만족. 이후 함수형 프로그래밍에 대한 더 많은 공부를 진행.
